# cwan

## 需要的環境

可對照 [這邊](https://tensorflow.google.cn/install/source_windows?hl=en#gpu) 相對應的版本去安裝
* python 3.6
* tensorflow-gpu 1.4
* CUDA 8.0
* cudnn 6.0
* numpy
* scipy
* matplotlib
* scikit_learn
